## Как обновить проект в Xcode

### Шаг 1: Заменить старые Swift файлы

Перетащи все 14 .swift файлов из архива в Project Navigator поверх старых.
Xcode спросит → выбери "Replace".

ВАЖНО: файл Stubs.swift из Сессии 1 — удали его (правый клик → Delete).

### Шаг 2: Добавить иконку приложения

1. Открой Assets.xcassets в Xcode
2. Удали существующий AppIcon
3. Перетащи папку AppIcon.appiconset из архива в Assets.xcassets

### Шаг 3: Добавить разрешение на камеру

Target → Info → добавить строку:
- Key: Privacy - Camera Usage Description
- Value: Для съёмки экрана станка или детали

### Шаг 4: Cmd+Shift+K (Clean) → Cmd+R (Run)

## Как пользоваться новыми функциями

### Переключение языка в чате
1. Настройки → выбери язык (RU/SK/EN/UA) → Сохранить
2. Следующее сообщение в чате — на новом языке
3. Системный промпт содержит инструкцию "отвечай ТОЛЬКО на этом языке"

### Анализ программы
1. Вкладка "Анализ программы"
2. Вставь G-код → "Проанализировать"
3. Появятся кнопки "Улучшить код" и "Что улучшить"
4. Внизу — поле для follow-up вопросов по этой программе

### Фото анализ
Вкладка "Ещё" → "Фото анализ" → Камера или Галерея → опц. вопрос → Анализировать

### Smart pills
Используй алармы/коды через быстрые кнопки — они запоминают частые и поднимают наверх

### Профиль станка
Настройки → Профиль станка → заполни → используется в анализе программы автоматически
